# 小屏手机数据库

真正的“小屏手机”爱好者追求的不是屏幕尺寸，而是能轻松塞进牛仔裤口袋的**小机身**。如果一台大屏手机拥有足够高的屏占比、整体尺寸足够紧凑，那它也值得被列入本列表。

因此，本数据库不以屏幕尺寸为单一标准，而是遵循 **“小屏手机 = 小尺寸手机”** 的理念，收集并整理了自 2017 年以来市面上出现过的各类紧凑型智能手机。

## 如何提名收录？

欢迎通过 Pull Request 直接修改 `data.csv` 文件来补充或更正数据。  

- 当前提名标准
  - 直板机：机身宽度 ≤ 72mm 重量 ≤ 200g
  - 折叠机：机身宽度 ≤ 80mm，重量 ≤ 220g
 

数据以 CSV 格式存储，包含以下字段（按顺序）：

| 字段     | 说明            |
| -------- | --------------- |
| 型号     | 手机完整名称    |
| 品牌     | 制造商          |
| 屏幕比例 | 如 16:9, 19.5:9 |
| 屏幕尺寸 | 对角线英寸数    |
| 宽度     | 机身宽度（mm）  |
| 厚度     | 机身厚度（mm）  |
| 重量     | 机身重量（g）   |
| 电池     | 电池容量（mAh） |
| 年份     | 发布年份        |
| SoC      | 处理器型号      |
| 类型     | 直板机 / 折叠机 |
